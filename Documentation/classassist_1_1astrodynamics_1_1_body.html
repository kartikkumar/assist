Detailed Description

This class functions as a container for body properties. At the moment, the state of a body can be stored in this class, and accompanying set- and get-functions are provided to use in conjunction with acceleration models and the Cartesian state derivative model in Tudat.

See Also
AccelerationModel, CartesianStateDerivativeModel.
